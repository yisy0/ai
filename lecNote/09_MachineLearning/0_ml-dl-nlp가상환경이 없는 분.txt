ml-dl-nlp 가상환경이 충돌나거나 없을 경우
3, 6, 7, 11 단계 다소 시간 소요
1 .jupyter 커널 목록에서 삭제 하기 : jupyter kernelspec remove ml-dl-nlp

2. 가상환경 지우기 : conda env remove --name ml-dl-nlp

3. 가상환경 생성 : conda create -n ml-dl-nlp python=3.10

4. 생성한 가상환경으로 전환 : conda activate ml-dl-nlp 

5. pip 최신버전으로 업데이트 : python.exe -m pip install --upgrade pip

6. pip install jupyter numpy==1.23.5 pandas==1.5.3  matplotlib==3.7.0 seaborn==0.12.2 scikit-learn==1.2.1

7. 가상환경에서 Jupyter를 설치해야 함 : conda install ipykernel

8. 가상환경을 Jupyter에서 선택 가능한 커널로 등록 : python -m ipykernel install --user --name ml-dl-nlp --display-name "ml-dl-nlp(ipykernel)"

8. pip install opencv-python==4.8.1.78
