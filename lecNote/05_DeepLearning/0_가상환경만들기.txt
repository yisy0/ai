※ 가상환경 만들기(ml-dl-nlp)

(가상환경 목록 확인) conda env list (conda info --envs)

(가상환경생성:-n:--name을 의미) conda create -n ml-dl-nlp python=3.10

(생성한 가상환경으로 전환) conda activate ml-dl-nlp

(pip 최신버전으로 업그레이드) python -m pip install --upgrade pip

(lib들 설치) pip install jupyter numpy==1.23.5 pandas==1.5.3  matplotlib==3.7.0 seaborn==0.12.2 scikit-learn==1.2.1

pip install tensorflow==2.10.0

(가상환경에서 jpykernel 설치) conda install ipykernel

(가상환경을 jupyter에서 선택가능한 커널로 등록)
python -m ipykernel install --user --name ml-dl-nlp --display-name "ml-dl-nlp(ipykernel)"

(가상환경 등록 확인)  jupyter kernelspec list

(가상환경을 jupyter 커널 목록에서 삭제 하기) jupyter kernelspec remove 가상환경이름(ml-dl-nlp)
(가상환경 지우기) conda env remove --name 가상환경이름
